# Barbería Alura :barber:

Bienvenido a la página web de la Barbería Alura, donde la elegancia y la profesionalidad se encuentran. Este sitio está diseñado para proporcionar una experiencia en línea intuitiva y atractiva para nuestros clientes, con una interfaz moderna y funcional.

## Descripción

La página presenta un diseño limpio y profesional con un menú de navegación intuitivo, un formulario de contacto eficiente y una galería de productos bien organizada. La página de inicio ofrece una vista general de los servicios, mientras que las páginas de productos y contacto proporcionan detalles adicionales y formas de interactuar con la barbería.

## Tecnologías Utilizadas :hammer:

- ![HTML5](https://img.shields.io/badge/-HTML5-E34F26?style=flat-square&logo=html5&logoColor=white) **HTML**: Estructura básica de las páginas.
- ![CSS3](https://img.shields.io/badge/-CSS3-1572B6?style=flat-square&logo=css3&logoColor=white) **CSS**: Estilos y animaciones para una presentación visual coherente.
- ![JavaScript](https://img.shields.io/badge/-JavaScript-F7DF1E?style=flat-square&logo=javascript&logoColor=black) **JavaScript**: Interactividad y funcionalidades dinámicas, como la validación del formulario.

## Funcionalidades

- **Diseño Adaptativo**: Diseño responsivo que se ajusta a diferentes tamaños de pantalla para una experiencia óptima en dispositivos móviles y de escritorio.
- **Menú de Navegación**: Menú interactivo con estilos mejorados, bordes redondeados y sombras para facilitar el acceso a las diferentes secciones del sitio.
- **Galería de Productos**: Sección de productos con una disposición de cuadrícula para una visualización clara, con efectos de hover para resaltar productos.
- **Formulario de Contacto**: Formulario para que los clientes puedan ponerse en contacto fácilmente, con validación de campos y estilos mejorados para inputs y botones.
- **Página de Reservación**: Formulario de reservación con inputs estilizados y botones con efectos visuales modernos.
- **Footer Consistente**: Footer unificado en todas las páginas para una navegación coherente, con estilos mejorados y ajustes para diferentes dispositivos.

## Uso

1. Clona este repositorio:

```sh
git clone https://github.com/CriXDZ/Barberia-Alura.git
```

2. Navega a la carpeta del proyecto:

```sh
cd Barbería-Alura
```

3. Abre `index.html` en tu navegador para ver la página de inicio.
